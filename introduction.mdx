# CyberScripts

CyberScripts is a collection of battle-tested shell scripts built for blue team competitors in cybersecurity competitions like CCDC and NCAE. The suite covers every phase of competition play: initial hardening, parallel deployment to multiple hosts, file integrity monitoring, C2 detection, and system baselining.

## How it works

The scripts are organized around the competition timeline:

<Steps>
  <Step title="Pre-competition">
    Set up your hostfile with IP addresses, system names, and pre-generated password hashes. Run `backup.sh` locally to save a clean baseline.
  </Step>
  <Step title="Competition start (first 15 minutes)">
    Run `minzero/deploy.sh` to push hardening scripts to all hosts in parallel. This sets firewalls, locks user accounts, and deploys your admin user across the entire range simultaneously.
  </Step>
  <Step title="Activation">
    SSH into each host and run `activate.sh` to deploy auditd rules, start Watchdawg, install BusyBox, and harden the SSH config.
  </Step>
  <Step title="Ongoing monitoring">
    Run `c2scanner.sh` or `angryc2scanner.sh` to watch for new outbound connections. Use `baseline/standard.sh` for an interactive security audit at any point during the competition.
  </Step>
</Steps>

## Requirements

| Dependency | Used by | Install |
|---|---|---|
| `sshpass` | `minzero/deploy.sh` | `apt install sshpass` |
| `parallel` | `minzero/deploy.sh` | `apt install parallel` |
| `auditd` | `activate.sh`, `auditkey-notifier.sh` | auto-installed by `activate.sh` |
| `iptables` | `minzero/autofirewall.sh` | pre-installed on most distros |
| `nc` (netcat) | `auditkey-notifier.sh` | `apt install netcat` |
| `curl` | `activate.sh`, `auditkey-notifier.sh` | pre-installed on most distros |

<Note>
All scripts are written in POSIX `sh` (with `bash` used only where noted) and are tested on Alpine, Debian/Ubuntu, and RHEL/CentOS. The `funcs/detectOS.sh` helper auto-detects the package manager at runtime.
</Note>
